EcoTrack

EcoTrack is a smart domestic waste management platform that enables citizens to report waste issues, participate in community discussions, purchase recycled products, and receive AI-powered assistance.

Features
- Waste reporting with images & location
- Community cleanliness posts
- AI chatbot for waste guidance
- Eco-store for recycled products
- Interactive map & dashboards

Tech Stack
- React + Tailwind + shadcn/ui
- Node.js + Express
- MongoDB
- AI APIs
- Maps integration
